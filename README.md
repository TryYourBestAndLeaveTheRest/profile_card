# Profile Card

This folder contains a responsive profile card component built with HTML, CSS, and vanilla JavaScript.

## Features

- **Semantic HTML**: The card is built with semantic HTML5 tags for better structure and accessibility.
- **Dynamic Time Display**: Shows the current time in milliseconds, updating every second.
- **Required `data-testid` Attributes**: All specified elements include `data-testid` attributes for automated testing.
- **Social Links**: A list of social media links that open in a new tab.
- **Responsive Design**: The layout adapts to different screen sizes, from mobile to desktop.
- **Accessibility**: The avatar image includes descriptive alt text.
